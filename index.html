animation-fill-mode

Definición y uso

animation-fill-mode es una propiedad de CSS que define cómo debe verse un elemento antes de que comience una animación y después de que termine.

  • Antes de que comience la animación: Esto se refiere al estado del elemento desde que se carga la página hasta que la animación empieza.
  • Después de que termine la animación: Esto se refiere al estado del elemento después de que la animación ha completado su ciclo.

Valores de animation-fill-mode

Valor Descripción
None El estilo no cambia
forwards El elemento mantiene el último estilo
backwards El elemento toma el primer estilo
both Afecta tanto antes como después

Ejemplos Didácticos

animation-fill-mode: forwards;
